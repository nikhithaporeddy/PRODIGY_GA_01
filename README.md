# PRODIGY_GA_01 - Generative AI Text Generation

## 🚀 Project Overview
This project demonstrates text generation using GPT-2 with custom input prompts.  
It generates meaningful and diverse outputs based on given input text.

## 🧠 Features
- GPT-2 based text generation
- Uses prompt-based approach (no fine-tuning required)
- Implements sampling techniques:
  - Top-k sampling
  - Top-p (nucleus sampling)
  - Temperature control
- Generates different outputs on each run
- Clean and error-free implementation

## 🛠️ Technologies Used
- Python
- PyTorch
- Hugging Face Transformers
